Introduction

Welcome to the VoiceForge AI documentation. Our API lets you convert text into natural, human-like speech using our neural voice engine — perfect for apps, podcasts, IVR systems, and more.

This guide covers how to authenticate, call our endpoints, and handle responses.

Quick Start

Here's the fastest way to generate your first audio clip using the /api/tts endpoint:

# Generate speech from text
curl -X POST https://your-domain.com/api/tts \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello from VoiceForge AI!",
    "voiceId": "your-voice-id",
    "speed": "normal"
  }' \
  --output speech.wav

You'll get back a .wav audio file that you can play or save directly.

Authentication

Account-specific endpoints, such as /api/history and /api/user/profile, require a login token. After signing in via /api/auth/login, include the returned token in your requests:

Authorization: Bearer YOUR_TOKEN_HERE
GET

/api/voices

Returns the full list of available voices, fetched live from our voice engine.

{
  "voices": [
    { "id": "voice_123", "name": "Aria", "language": "en" },
    ...
  ]
}
POST

/api/tts

Generates a speech audio file from the given text and returns raw WAV audio bytes.

Body Parameters

ParameterTypeRequiredDescription
textstringYesThe text to convert to speech
voiceIdstringYesID of the voice to use
speedstringNo"slow", "normal", or "fast"
emotionstringNo"excited", "content", "sad", "angry"
laughterbooleanNoAdd a laughter sound at the end
GET

/api/history

Returns the authenticated user's past voice generations, sorted by newest first. Requires the Authorization header.

Error Handling

All errors are returned as JSON with a descriptive message and appropriate HTTP status code:

{
  "error": "text and voiceId are both required"
}

Rate Limits

Rate limits depend on your subscription plan — see the Pricing page for character limits per plan. Exceeding your monthly limit will pause generation until your next billing cycle or upgrade.
